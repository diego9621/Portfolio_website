AI Product Matching Algorithm

Developed a sophisticated two-stage matching system combining cosine similarity vector embeddings with GPT-4 intelligence to accurately match products across competitor catalogs with over 95% accuracy.

The system processes cleaned catalog data, generates vector embeddings for semantic similarity, identifies top-k candidates, and leverages AI to select the most confident matches while rejecting uncertain ones.

Technical Deep Dive

Async Scraping Architecture

Built high-performance async scrapers using aiohttp with intelligent concurrency limits (5-50 requests) to optimize speed while respecting server constraints and avoiding blocks.

Intelligent Data Cleaning

Implemented multi-strategy cleaning pipeline with "safe" and "aggressive" modes, handling abbreviation expansion, regex patterns, and competitor-specific data normalization.

Real-time BigQuery Integration

Seamless data pipeline exporting cleaned competitor data to BigQuery with competitor-specific field mapping and automated table management for immediate analytics access.
